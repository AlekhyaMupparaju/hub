# Node.js Starter

## Requirements
* [node.js v0.10.x](http://nodejs.org/download/)
* [npm 1.1.x](http://npm.org)
* [MongoDB](http://docs.mongodb.org/manual/installation/)

## Installation

Run from your terminal:

    git clone git@github.com:gravityonmars/nodejs-boilerplate.git
    cd nodejs-starter
    make 

## Test

    npm test
